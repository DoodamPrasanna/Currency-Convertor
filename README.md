# 💱 ExchangePro

A modern and responsive currency converter built using HTML, CSS, and JavaScript.

## 🚀 Features

- Real-time currency conversion
- Modern dark glassmorphism UI
- Currency swap functionality
- Live exchange rates
- Conversion history
- Responsive design
- Country flags support
- Smooth animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API
- FlagCDN

## 📸 Preview

ExchangePro provides accurate currency conversions with a sleek and professional interface.
